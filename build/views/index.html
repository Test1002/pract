Проект на практику

  • О проекте
  • О практиканте
  1. Создать на портале Github тестовый пустой репозиторий.
  2. Установить на машине разработчика систему управления версиями Git, настроить ее для работы с репозиторием тестового проекта на Github.
  3. Создать тестовый проект на основе Nodejs, в качестве базового фреймворка использовать Express js.

    4. Тестовый проект должен содержать два обработчика GET HTTP запросов:

    * / - обработчик корневого запроса должен выводить страницу описывающую содержание проекта.

    * /about - обработчик запроса должен выводит данные о практиканте.

    Верстка страниц должна быть выполнена на языке Jade. Сборка осуществляться с помощью Grunt js.

  4. Сохранить тестовый проект в Github репозитории.
  5. Создать проект на Heroku.
  6. Настроить тестовый проект для развертывание на Heroku.
  7. Развернуть систему управления сборками Jenkins
  8. Настроить Jenkins на выполнение следующих задач:

    9. * Выгрузка тестового проекта с github

    * Сборка тестового проекта с помощью grunt js

    * Развертывание тестового проекта на heroku
